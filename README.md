# Kakao Skills

Public repository for KakaoTalk-related Codex skills.

## Included skills

- `kakao-chat`: Read visible KakaoTalk conversations or send user-confirmed KakaoTalk messages on macOS or Windows desktop environments.

## Notes

This skill operates the KakaoTalk desktop UI. It does not use a KakaoTalk API. It includes separate guidance for macOS accessibility automation, Windows Codex Computer Use, and Windows PowerShell fallback paths.
